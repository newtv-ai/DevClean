# Security policy

Reclaimer is pre-alpha and inventory-only. Do not use unreleased builds to authorize deletion.

Please report suspected path traversal, reparse-point traversal, data modification during scan,
credential exposure, or privilege-boundary defects privately to the project owner. Do not attach
sensitive paths, tokens, or private registry URLs to public reports.

Any confirmed unintended mutation or action outside an adapter-approved root blocks release and
requires disabling the affected capability.
