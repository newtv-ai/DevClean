# Changelog

本项目遵循语义化版本。Pre-Alpha 阶段的接口仍可能变化。

## [Unreleased]

### Added

- 同会话 `ReadDirectoryChangesW` 增量刷新与不确定时全量回退。
- 有界 AI 复核导出/导入；建议无执行权限，需用户显式采纳。
- 最多 256 文件的用户精确计划，自动拆成每批最多 32 项的 SQLite 意图批次。
- 可恢复同卷私有隔离、隔离区查看/恢复和独立强确认永久清除。
- Windows 句柄绑定身份复验、批准根最终路径约束和运行期用户资产/系统路径拒绝清单。
- 来源域、细分类、队列、执行上限、风险、证据、恢复能力和空间概览。
- 有界重复文件分析和执行后增量核验。

### Changed

- AI 由必经复核改为可选辅助；允许用户直接选择本地执行策略支持的候选。
- 浏览器/VS Code/缩略图缓存从系统只报告策略中拆出，进入人工/可选 AI 复核。
- 隔离目录改为批准根直属、随机批次名、原子私有创建；预存在路径不再接管。
- 隔离不再改写源文件 DACL，恢复后保留原文件权限。
- GUI 扫描状态改为仅保存在当前进程内存中，不再携带未接入产品的全盘
  inventory SQLite；小型执行安全日志只轮转已完成历史，未决状态始终保留。
- Windows EXE 构建会同时生成 CPython、Tcl/Tk、PyInstaller 与项目许可证
  目录，缺失任一必需文本即构建失败。

### Removed

- 扫描时自动永久删除。
- AI 直接永久删除路径。
- 可能退化成永久删除的 Windows Shell 回收站桥。
- 旧的未受控 `recycle` CLI 和相关执行模块。

### Security

- 已撤回曾在扫描阶段删除文件的预发布构建；详情见 [SECURITY.md](SECURITY.md)。
