# Community Code of Conduct

DevClean 社区欢迎不同经验、背景和观点的参与者。讨论应围绕事实、复现步骤、风险和用户结果展开。

参与者应：

- 尊重他人，不进行骚扰、歧视、威胁、羞辱或人身攻击；
- 对安全问题保持克制，不催促他人运行未经验证的破坏性代码；
- 清楚区分事实、推断、建议和个人偏好；
- 在发现错误时及时更正，并保护报告者和受影响用户的隐私；
- 对代码、文档、测试和设计意见给予具体、可执行的反馈。

维护者可以编辑或移除违反本规范的内容，临时或永久限制参与，并在合理范围内说明处置原因。安全事件或行为问题请通过 [SECURITY.md](SECURITY.md) 中的私下渠道报告。维护者会尽量保密、公平处理并避免报复。
